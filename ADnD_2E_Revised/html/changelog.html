Announcements

Changes in

  • Added buttons do deal critical damage, to the Player's Option weapons. This follows the Critical Hits: System I from the PO: Combat & Tactics
  • Added buttons do deal critical damage and effect, to the Player's Option weapons. This follows the Critical Hits: System II from the PO: Combat & Tactics
  • Added fields in the repeating Player's Option weapons damage and ammo for Size, and Type, used for Crit System II
  • Added fields in the repeating spells section for Subtlety, Sensory, Knockdown, and Critical Size
  • Added button to do critical effect, to the repeating spells section.
  • Added Special Talents from the PO: Combat & Tactics book to the Proficiency tab, with autofill
  • Added Special Talents to the Character Point tracker on the Character -> Info -> Tracker page
  • Added Traits from the PO: Skills & Powers book to the Proficiency tab, with autofill
  • Added Disadvantages from the PO: Skills & Powers book to the Proficiency tab, with autofill
  • Added Nonweapon proficiencies from the PO: Skills & Powers book
  • Added Weapons from The Complete Wizard's Handbook
  • Added Weapons from Player's Option: Combat & Tactics
  • Added Weapons from Player's Option: Skills & Powers
  • Added Spells from the PO: Spells & Magic book
  • Added Lorloveim's Creeping Shadow from Tome of Magic
  • Fixed a lot of bugs with various spells.
  • Fixed additional monster attacks, not being added to the turn tracker, when rolling initiative
  • Updated design for Potions, Dusts, and Scrolls (no fields have been added or removed. Only moved around)
  • Added Sheet Setting to enable the rearranged Priest Spheres from the Player's Option: Spells & Magic book
    • Changes can be seen here: https://htmlpreview.github.io/?https://github.com/Roll20/roll20-character-sheets/blob/master/ADnD_2E_Revised/misc/spheres-overview.html
  • Added Sheet Setting to enable the rearranged Wizard Schools from the Player's Option: Spells & Magic book
    • Changes can be seen here: https://htmlpreview.github.io/?https://github.com/Roll20/roll20-character-sheets/blob/master/ADnD_2E_Revised/misc/schools-overview.html
  • Reworked weapons autofill to support different version of weapons, to accommodate differences between books. If a given weapon have differences between the books you have activated on your sheet, you will be asked to pick what version to use.
    • For the data nerds the raw data can be seen at this link below:
    • https://htmlpreview.github.io/?https://github.com/Roll20/roll20-character-sheets/blob/master/ADnD_2E_Revised/misc/weapons-overview.html
    • If any data is wrong, please check the that you have the newest printing of the given book, and feel free to reach out, so it can get it corrected :)
  • Reworked nonweapon proficiency autofill to support different version of nonweapon proficiencies, to accommodate differences between books. If a given nonweapon proficiency have differences between the books you have activated on your sheet, you will be asked to pick what version to use.
    • For the data nerds the raw data can be seen at this link below:
    • https://htmlpreview.github.io/?https://github.com/Roll20/roll20-character-sheets/blob/master/ADnD_2E_Revised/misc/nonweapon-proficiencies-overview.html
    • If any data is wrong, please check the that you have the newest printing of the given book, and feel free to reach out, so it can get it corrected :)
